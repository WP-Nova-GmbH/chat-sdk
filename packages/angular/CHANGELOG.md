# @wp-nova/chat-sdk-angular

## 1.0.1

### Patch Changes

- 3192cea: Publish the Angular wrapper from its built `packages/angular/dist` directory so
  the package is resolvable from npm. The `1.0.0` Angular tarball was published from
  the package root, whose manifest has no `main`/`module`/`types`/`exports` (those
  are generated by ng-packagr into the dist manifest), so
  `npm install @wp-nova/chat-sdk-angular` failed with `ERR_MODULE_NOT_FOUND`. The
  release script now publishes `packages/angular/dist` and a new
  `check-angular-publishable` guard verifies the built artifact declares its entry
  points before publishing.
